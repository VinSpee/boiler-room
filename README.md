## INSTALLATION

```bash
npm i
```

## USE

### to develop
```bash
npm start
```
### to build
```bash
npm run build
```
### to clean
```bash
npm run clean
```

### to change namespace
```js
"package.json": {
	"config": {
		"namespace": "your-namespace"
	}
}
```
